Présentation de l'application :

Contexte :

Le port de plaisance Russell est un lieu dynamique accueillant une grande variété de bateaux. Afin de simplifier et d'optimiser la gestion des réservations des catways (petits appontements utilisés pour amarrer les bateaux), une application web a été développée pour faciliter le processus de réservation, d'administration et de suivi des amarrages.
Cette application est accessible via une interface web sécurisée, utilisant une API privée pour assurer la confidentialité et la sécurité des données des utilisateurs et des réservations.

Fonctionnalités prinpales de l'API nécessaires à la capitainerie :

Pour la gestion des catways :

  • Créer un catway,
  • Lister l'ensemble des catways,
  • Récupérer les détails d'un catway en particulier,
  • Modifier la description de l'état d'un catway en particulier,
  • Supprimer un catway,

Pour la gestion des réservations :

  • Prendre la réservation d'un catway,
  • Supprimer une réservation,
  • Lister l'ensemble des réservations,
  • Afficher les détails d'une réservation en particulier.

API sécurisée et Authentification :

  • L'application repose sur une API privée permettant de communiquer en toute sécurité avec la base de données.
  • Authentification JWT (JSON Web Token) : Les utilisateurs et les administrateurs doivent se connecter via un système d'authentification sécurisé pour accéder à certaines fonctionnalités. Un token JWT est généré et utilisé pour valider les actions des utilisateurs.

Technologies utilisées :

  • Frontend : l'application web utilise des langages HTML, CSS et JavaScript pour offrir une interface utilisateur fluide et interactive.
  • Backend : l'API est construite en Node.js avec le framework Express.js, offrant une gestion efficace des routes et des requêtes.
  • Base de Données : La base de données est gérée avec MongoDB, offrant une solution flexible et scalable pour stocker les informations des catways et des réservations.
  • Sécurité : l'application utilise des mécanismes de sécurité robustes pour protéger les données des utilisateurs, notamment des tokens JWT pour l'authentification et la gestion des sessions.

Accessibilité et Utilisation :

  • L'interface est accessible depuis n'importe quel appareil connecté à Internet (ordinateur, tablette, smartphone).
  • Le tableau de bord de l'application permet aux utilisateurs d'effectuer toutes les actions nécessaires à la gestion des réservations, avec une interface simple et intuitive.
