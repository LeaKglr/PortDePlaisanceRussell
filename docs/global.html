(async) add(req, res) → {Object}

## Vue d'ensemble Cette fonction permet de créer une réservation associée à un catway spécifique en utilisant l'ID du catway fourni dans les paramètres de l'URL. Les informations de la réservation sont passées dans le corps de la requête. Si le catway existe, la réservation est ajoutée et renvoyée.
Parameters:
Name Type Description
req Object La requête HTTP.
Properties
Name Type Description
params Object Les paramètres de la requête.
Properties
Name Type Description
id string L'ID du catway pour lequel la réservation est créée.
body Object Les informations de la réservation.
Properties
Name Type Description
clientName string Le nom du client.
boatName string Le nom du bateau.
checkIn string La date d'arrivée.
checkOut string La date de départ.
res Object La réponse HTTP.
Source:
  • reservationservice.js, line 4
Throws:
- Si une erreur se produit lors de la création de la réservation. ## Tutoriel 1. Envoyer une requête POST à l'API avec l'ID du catway dans les paramètres de l'URL. 2. Inclure les informations dans le corps de la requête : - `clientName`: Le nom du client. - `boatName`: Le nom du bateau. - `checkIn`: La date d'arrivée (format ISO 8601 recommandé). - `checkOut`: La date de départ (format ISO 8601 recommandé).
Type
Error
Returns:
- Retourne l'objet de la réservation créée.
Type
Object
Example
Exemple de requête avec cURL :
```bash
POST http://localhost:3001/api/catways/60b8c89b5414c5195ca65d99/reservations
Content-Type: application/json

{
  "clientName": "Jean Dupont",
  "boatName": "L'Hirondelle",
  "checkIn": "2025-01-15T10:00:00Z",
  "checkOut": "2025-01-20T10:00:00Z"
}
```

Exemple de réponse réussie :
```json
{
  "_id": "60c8f89a5414c5195ca67d80",
  "catwayNumber": 123,
  "clientName": "Jean Dupont",
  "boatName": "L'Hirondelle",
  "checkIn": "2025-01-15T10:00:00Z",
  "checkOut": "2025-01-20T10:00:00Z"
}
```

Si le catway n'existe pas, une erreur 404 est renvoyée :
```json
{
  "message": "Catway non trouvé."
}
```

## Glossaire :
- Catway : petit appontement pour amarrer un bateau.
- catwyaNumber : numéro de pont.

Properties

res Object La réponse HTTP.
Source:

Throws:
- Si une erreur se produit lors de la suppression. ## Tutoriel 1. Envoyer une requête DELETE à l'API avec les IDs du catway et de la réservation dans les paramètres de l'URL.
Type
Error
Returns:
- Retourne un message de succès ou une erreur.
Type
Object
Example
Exemple de requête avec cURL :
```bash
DELETE http://localhost:3001/api/catways/60b8c89b5414c5195ca65d99/reservations/60b9c89b5414c5195ca67e01
```

Exemple de réponse réussie :
```json

}
```

```json
{
  "message": "Catway non trouvé."
}
```
ou
```json
{
  "message": "Réservation non trouvée."
}
```
